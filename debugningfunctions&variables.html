let vs const vs var

  • var: old keyword but still work. it is preferable to use let and const the reason has to do with scope as var has a function scope while the latter have block scope
  • const : can not be reasigned and it needs to be have a value right away
  • let can be reasigned

    • IMPLICIT TYPE CONVERSION
    js converts implicitly the strings that contain a number to a number type when doing arithmetic operations

    EXAMPLE: const num1 = '34'
    let decimal1 = '3.5'
    let value = num1 - decimal1

    it works well on most of arithmetics operation but on addition it will concatinate and not add
